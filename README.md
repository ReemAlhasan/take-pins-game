# take-pins-game
A simple strategy game where players take pins from a board

## About the Game
Take Pins (also known as Nim) is a two-player strategy game. Players take turns removing 1 or 2 pins from a board that starts with 20 pins. The player who takes the last pin wins the game.

## Game Rules
- The board starts with 20 pins
- Players alternate turns
- On each turn, a player must take 1 or 2 pins
- The player who takes the last pin wins
